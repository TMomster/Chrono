# Chrono: An Elegant Timer Application

中文用户参阅 [ReadmeCN.md](ReadmeCN.md)

## Overview

Chrono is a versatile and user-friendly timer application designed to provide an intuitive and visually appealing timing experience. With features such as theme switching, language support, and customizable controls, Chrono aims to be both functional and aesthetically pleasing.

## Features

- **Theme Switching**: Toggle between light and dark themes with smooth transitions.
- **Language Support**: Switch between Chinese and English.
- **Customizable Controls**: Start, pause, and reset the timer using either mouse clicks or keyboard shortcuts.
- **Draggable Window**: Move the window freely by dragging the title bar.
- **High Precision Timing**: Timing accuracy up to 0.01 seconds.
- **Hour Display**: Supports displaying hours when the timer exceeds 60 minutes.
- **About Window**: Displays version information and usage instructions.

## Usage

### Starting and Pausing the Timer

- **Mouse**: Click on the time display area or the "Start/Pause" button.
- **Keyboard**: Press the `Enter` key.

### Resetting the Timer

- **Mouse**: Click the "Reset" button when the timer is paused.
- **Keyboard**: Press the `Space` key or `ESC` key when the timer is paused.

### Theme and Language Switching

- **Mouse**: Click on the "More" button, then select the desired theme or language from the menu.
- **Keyboard**: Press the `R` key to toggle between light and dark themes.

### Window Operations

- **Minimize**: Click the minimize button (—) in the title bar.
- **Close**: Click the close button (×) in the title bar.
- **Emergency Close**: Press the `F4` key to close the application immediately.

### Displaying the About Window

- **Mouse**: Click on the "About" option in the "More" menu.
- **Keyboard**: Press the `H` key.

## Keyboard Shortcuts

- **Start/Pause Timer**: `Enter`
- **Reset Timer**: `Space` (when paused)
- **Toggle Theme**: `R`
- **Show About Information**: `H`
- **Emergency Close**: `F4`
